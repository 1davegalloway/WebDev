A Weather Analysis Nearing the Equator

"How does the weather change when nearing the equator?"

Latitude vs. Weather

The purpose of this project was to analyze how weather changes as you get closer to the equator.

To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

Afer assembling the dataset, we used Matplotlib to plot various aspects of the eather vs. latitude. Factors included: temperature, cloudiness, wind speed, and humidity. This site also provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations found.
